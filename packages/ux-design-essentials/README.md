# UX Design Essentials

Kompaktes Wissen über gutes Interface-Design. 6 Dokumente die dein Agent liest, wenn du UI-Arbeit machst.

## Was drin ist

- **HEURISTICS** — 10 Nielsen-Regeln mit Beispielen
- **ANTI_PATTERNS** — was NIE in UI tun
- **REFERENCE_APPS** — Benchmarks pro Screen-Typ (Raycast, Linear, Stripe…)
- **INFORMATION_ARCHITECTURE** — Navigation, Tiefe, Admin-Trennung
- **COPY_GUIDE** — Button-Labels, Empty States, Error-Messages
- **PERSONA_FRAMEWORK** — eigene Zielgruppe für UI-Entscheidungen schärfen

## Wie nutzen

Nach Install triggert das Paket bei Stichwörtern wie „UI bauen", „Screen designen", „Empty State", „Copy prüfen". Der Agent zieht die relevanten Dateien als Kontext.

Kombiniere mit deinen eigenen Brand-Files (CI.md, Brand-Voice.md, deine Persona) für on-brand Output der auch UX-mäßig gut ist.

## Markenneutral

Enthält KEINE spezifischen Farben, Fonts oder Tonalitäten. Nur generische Prinzipien. Dein Brand bleibt deins.
